# 📝 TextUtils

TextUtils is a simple and user-friendly React application that helps users analyze and transform text quickly. It provides a collection of useful text editing tools in one place, making everyday text processing faster and more convenient.

## 🚀 Features

* ✨ Convert text to **UPPERCASE**
* ✨ Convert text to **lowercase**
* ✨ Capitalize the first letter of each sentence
* ✨ Copy text to the clipboard
* ✨ Clear all text with one click
* ✨ Remove extra spaces
* ✨ Count words and characters
* ✨ Estimate reading time
* ✨ Live text preview
* 🌙 Light & Dark Mode
* 📱 Responsive design for desktop and mobile devices

## 🛠️ Built With

* React.js
* JavaScript (ES6)
* Bootstrap 5
* HTML5
* CSS3
